Esse projeto foi desenvolvido na Fatec Rio Claro para a disciplina Estrutura de Dados. O objetivo do projeto foi entender as estruturas de dados e programação orientada a objetos.

O projeto consiste em um sistema de gerenciamento de álbum de figurinhas da copa do mundo. O sistema permite:

  . Inserir figurinha no álbum
  . Remover figurinha do álbum pelo número da figurinha
  . Consultar figurinha pelo número
  . Buscar jogador
  . Buscar seleção
  . Vizualizar o álbum completo
  . Ver figurinhas repetidas
  . Ver a quantidade de repetidas
  . Ver a porcentagem do álbum concluído
  . Gerar 100 figurinhas falsas com a biblioteca faker para não ter que inserir apenas manualmente
  . Gerar o álbum completo com o faker para testar as funcionalidades do sistema
  . Salvar o álbum em arquivo json
  . Fazer trocas de figurinhas
  . Ver o histórico de trocas de figurinhas


  As estruturas de dados utilizadas foram a Lista Encadeada para gerenciamento do álbum e Fila para registrar o histórico de trocas. Não foram utilizadas estruturas built-in do Python como list e deque.


  Para rodar o projeto, primeiramente instale a biblioteca faker e execute o arquivo main.py
